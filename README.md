# Flour Hour üçû

A bread baking schedule planner that takes the guesswork out of timing.

Pick a recipe, set your start time or when you want to eat, and Flour Hour calculates every step with exact timestamps. No more mental math, sticky notes, or missed windows.

![Flour Hour Screenshot](./public/flour-hour-preview.png)

## Features

- **22 recipes** - From 30-minute discard pancakes to 36-hour croissants
- **Flexible scheduling** - Set your start time or work backwards from when you want to bake
- **Step-by-step timing** - Every step calculated to the minute
- **Sourdough & yeasted breads** - Classic loaves, focaccia, bagels, brioche, and more
- **Clean interface** - Multiple themes to choose from

```bash
cd flour-hour
npm install
npm run dev
```
